# Phase 2: 维度辩论

## 你的任务
完成当前维度的红蓝辩论

## 规则（4条）

1. **同时只讨论一个维度**
   - 当前维度：{{dimension_name}}
   - 不跳到其他维度

2. **红蓝裁判3角色并行**
   - 红队：攻击边界条件（至少3个攻击点）
   - 蓝队：逐条回应，构建方案
   - 裁判：评分（1-5分），至少1个≥3分

3. **轮次控制**
   - L1/L2：1轮（直接收敛边界）
   - L3：2轮（第1轮发散功能，第2轮收敛边界）

4. **输出结构化**
   - 攻击点列表（含评分）
   - 回应方案
   - 争议点记录

## 输入
- 维度主题
- 模板默认值
- 调研报告（按需回查）

## 产出
- dimension-{{n}}.json（结构化输出）

## 你不知道
- 其他维度的辩论内容
- 后面的审核裁决
- 最终规范
